# pludコマンド

#### コマンド概要
標準入力から読み込んだ数字を足す

#### 使用方法
```
$ seq [数値] | ./plus
```
#### 使用例
```
$ seq [5] | ./plus
```

* #### 出力結果
```
15
```

## 必要なソフトウェア
* Python
  * テスト済み: 3.7～3.10

## テスト環境
* Ubuntu
